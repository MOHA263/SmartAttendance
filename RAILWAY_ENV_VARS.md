# Railway Environment Variables Template

Copy these variable names and values into Railway dashboard.

## For Railway Internal Connection (Railway auto-links MySQL service)

```
SPRING_DATASOURCE_URL=jdbc:mysql://${{ MYSQLHOST }}:${{ MYSQLPORT }}/${{ MYSQLDATABASE }}?useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true
SPRING_DATASOURCE_USERNAME=${{ MYSQLUSER }}
SPRING_DATASOURCE_PASSWORD=${{ MYSQLPASSWORD }}
MAIL_USERNAME=your-gmail@gmail.com
MAIL_PASSWORD=your-app-password
PORT=8080
```

---

## What Each Variable Does

| Variable | What it is | Where to get it |
|----------|-----------|-----------------|
| `SPRING_DATASOURCE_URL` | MySQL connection string | Auto-generated by Railway |
| `SPRING_DATASOURCE_USERNAME` | DB login username | Auto-generated by Railway MySQL |
| `SPRING_DATASOURCE_PASSWORD` | DB login password | Auto-generated by Railway MySQL |
| `MAIL_USERNAME` | Your Gmail address | You provide |
| `MAIL_PASSWORD` | Gmail app password (16 chars) | Get from Gmail security page |
| `PORT` | Server port (usually 8080) | Default provided |

## Step-by-Step to Add Variables in Railway

### Method 1: Manual Input

1. Go to Railway Dashboard â†’ Your Project
2. Click on **SmartAttendance** service
3. Click **"Variables"** tab
4. Click **"Add Variable"** button
5. Type in left box: `SPRING_DATASOURCE_URL`
6. Type in right box: `jdbc:mysql://${{ MYSQLHOST }}:${{ MYSQLPORT }}/${{ MYSQLDATABASE }}?useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true`
7. Press Enter
8. Repeat for other 5 variables below

### Method 2: Copy-Paste (Easier)

1. Select all text below (between the dashes)
2. Copy to clipboard
3. In Railway Variables tab, paste

## How to Get Gmail App Password

1. Go to https://myaccount.google.com/security
2. **Enable 2-Factor Authentication** (if not already done)
3. Search for **"App passwords"** (in left menu)
4. Select:
   - **App:** Mail
   - **Device:** Windows Computer (or your OS)
5. Click **"Generate"**
6. Google shows 16-character password
7. Copy & paste as `MAIL_PASSWORD`

Example: `abcd efgh ijkl mnop`

## Railway MySQL Variables Auto-Generated

When you create MySQL service in Railway, these are automatically created:

- `MYSQLHOST` = railway.app hostname
- `MYSQLPORT` = 3306 (standard)
- `MYSQLDATABASE` = rail... (auto name)
- `MYSQLUSER` = root
- `MYSQLPASSWORD` = secure random password

Railway handles this automatically - no manual setup needed!
